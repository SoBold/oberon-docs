# Forgot Password Form

This form allows users to initiate a password reset by entering their email address.

---

## Example HTML Output

```html
<form id="obrnForgotForm">
    <div class="obrnLogin__FormGroup">
        <label for="user_email">Email Address</label>
        <input type="email" name="user_email" id="user_email" />
    </div>
    <div
        class="obrnLogin__Msg error"
        id="loginForgotMsg"
        style="display:none"
    ></div>
    <div class="obrnLogin__Submit">
        <button type="submit" name="submit">Send Reset</button>
        <a href="javascript:void(0);" id="obrnLogin__SignInTrigger"
            >Go back to sign in</a
        >
    </div>
</form>
```

---

## Behavior

-   **Form ID**: `#obrnForgotForm`
-   **Email Field**: Collects the user's email.
-   **Error Display**: Element with `id="loginForgotMsg"` shows validation or server messages.
-   **Reset Link**: Sends a password reset email.
-   **Back to Sign In**: Clickable element with `id="obrnLogin__SignInTrigger"` that should trigger UI toggle back to the login form.

---

## Usage

```php
[obrn-forgot-password-form]
```

Place this shortcode where users need access to the password recovery option.
