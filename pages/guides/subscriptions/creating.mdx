## How to create a subscription

### Construct a subscription controller

Construct a new instance of the subscription controller

```php
$subscriptionController = new \Oberon\Controllers\SubscriptionController();
```

### Create the subscription via the controller API

Use the method `createSubscription` on the SubscriptionController class to create an instance of the Oberon Subscription model. It will return back a new instance of the Oberon subscription model.

```php
$subscription = $subscriptioncontroller->createSubscription([
    {subscription_data}
]);
```
