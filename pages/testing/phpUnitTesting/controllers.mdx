## Test Controllers

### Description
This class tests the functionality and reliability of the controllers within the Oberon package.

### Setup
- Set up the parent test environment.
- Run the activation.

### Tests

#### `testMemberController`

- **Objective**: Test the functionality of the `MemberController`.

- **Actions**:
  - Create a member.
  - Fetch the created member.
  - Update the member's data.
  - Validate the updated data.
  - Delete the member.

- **Assertions**:
  - Verify that creating, fetching, updating, and deleting a member results in the expected outcomes.
