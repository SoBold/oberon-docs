# Subscription Controller

The Subscription Controller in the Oberon plugin manages subscription-related operations, such as creation, retrieval, updating, and deletion of subscriptions.

## Class: `SubscriptionController`

This class provides methods to interact with subscription data.

### Method: `createSubscription`

Creates a new subscription.

**Usage:**

```php
$subscriptionController = new SubscriptionController();
$newSubscription = $subscriptionController->createSubscription([
    'user_id' => 123,
    'membership_id' => 456,
    'period' => 12,
    'period_type' => 'months',
    'renew' => true,
    'status' => 'active',
]);
```

### Method: `getSubscription`

Retrieves subscription information by subscription ID.

**Usage:**

```php
$subscriptionController = new SubscriptionController();
$existingSubscription = $subscriptionController->getSubscription($subscription_id);
```

### Method: `updateSubscription`

Updates an existing subscription.

**Usage:**

```php
$subscriptionController = new SubscriptionController();
$updateSuccess = $subscriptionController->updateSubscription($subscription_id, [
    'period' => 6,
    'status' => 'suspended',
]);
```

### Method: `deleteSubscription`

Deletes a subscription.

**Usage:**

```php
$subscriptionController = new SubscriptionController();
$deleteSuccess = $subscriptionController->deleteSubscription($subscription_id);
```
