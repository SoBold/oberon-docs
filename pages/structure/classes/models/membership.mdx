# Membership Model

The Oberon Membership Model manages post meta values defined in the 'membership' post type and loads auxiliary data for the membership Model.

## Properties

| Property Name         | Type   | Description                                       | Version |
| --------------------- | ------ | ------------------------------------------------- | ------- |
| id                    | int    | The membership ID                                 | 0.1.0   |
| membership_meta_keys  | array  | Membership Meta Keys                              | 0.1.0   |
| title                 | string | Membership Title                                  | 0.1.0   |
| name                  | string | Membership Name                                   | 0.1.0   |
| type                  | string | Membership type                                   | 0.1.0   |
| group_id              | int    | Membership Group ID                               | 0.1.0   |
| group_order           | int    | Membership Group Order                            | 0.1.0   |
| billing_type          | string | Membership billing type                           | 0.1.0   |
| price                 | float  | Membership price                                  | 0.1.0   |
| membership_type       | string | Membership type                                   | 0.1.0   |
| period                | int    | Membership period                                 | 0.1.0   |
| period_type           | string | Membership period type                            | 0.1.0   |
| status                | string | Membership status                                 | 0.1.0   |
| rules                 | array  | Membership rules                                  | 0.1.0   |
| is_recurring          | bool   | Indicates if the membership is recurring          | 0.1.0   |
| enable_child_accounts | bool   | Indicates if the membership allows child accounts | 0.1.0   |
| max_child_accounts    | int    | Indicates max allows child accounts               | 0.1.0   |

## Methods

### `__construct`

**Parameters:**

-   `$id`: The membership ID or `0` to construct a new instance.

**Description:** Initializes the model. If no ID is provided or if the ID is not associated with a valid membership post type, the method returns without doing anything. Otherwise, it loads meta and auxiliary data.

**Version:** `0.1.0`

### `loadMeta`

**Description:** Loads the meta data for the membership post type. It retrieves meta values defined in the 'Oberon\WordPress\PostTypes\Memberships' class.

**Version:** `0.1.0`

### `loadAuxiliaryData`

**Description:** Loads auxiliary data for the Post Model. This includes data natively set by WordPress post type, data that is a derivative of original meta data, and data not originally set by the membership post type class but still exists in the membership post type class.

**Version:** `0.1.0`

### `periodToText`

**Description:** Converts a numerical period and period type into a readable text description.

**Version:** `0.1.0`

### `fields`

**Returns:** Array

**Description:** Retrieves fields for the form using the MetaHelper.

**Version:** `0.1.0`

### `getMembershipOptions`

**Returns:** Array

**Description:** Get membership list for upgrade/downgrade.

**Version:** `0.1.0`

## Usage

To construct a membership instance

```php
$membership = new \Oberon\Models\Membership(1);
```
